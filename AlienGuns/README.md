## 当前内容

-   法师之手(#0)
    -   神话生物的残片。
    -   会交替射出分散的<span style="color=orange">**火属性**</span>与相对集中的<span style="color=purple">**空间属性**</span>子弹。
-   反方向的筒(#1)
    -   在脚下产生爆炸，并将玩家推离地面。
-   MP-155 瘟疫(#2)
    -   射出随机属性的子弹；在击中敌人后分裂并部分弹射至周围敌人。
    -   子弹穿甲等级越高，可分裂的级数越多。
-   M107 风暴(#3)
    -   射出高穿透性，且可以在障碍物表面反射的<span style="color=purple">**空间属性**</span>光束。
-   智能MCX Spear(#4)
    -   "RTX ON!"
    -   火控系统全面升级的MCX Spear，在机瞄状态会标记准心附近的敌人，并使子弹具有一定的追踪性。使用武器或双击快捷切换键以轮换追踪子弹发射方式，若按住奔跑键进行则以相反方向轮换。
    -   另外，每5颗子弹与最后5颗子弹替换为了不同颜色的曳光弹，用于提示使用者剩余弹量。
-   CCB-ChickBombs(#5)
    -   老爷爷，我们来给你踩踩背啦！
