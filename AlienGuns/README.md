# Alien Guns 外星枪械

一堆什么都能射，就是射不了常规子弹的枪；???最有用的一集

_注：本 mod 所有物品 id 号段从`"YukkuriC".GetHashCode() = 1206528349`开始；以下标注 id 均为此基础上的偏移量_

## 当前内容

-   法师之手(#0)
    -   神话生物的残片。
    -   会交替射出分散的 **火属性** 与相对集中的 **空间属性** 子弹。
-   反方向的筒(#1)
    -   在脚下产生爆炸，并将玩家推离地面。
-   MP-155 瘟疫(#2)
    -   射出随机属性的子弹；在击中敌人后分裂并部分弹射至周围敌人。
    -   子弹穿甲等级越高，可分裂的级数越多。
